- ```java
  ArraysCopyOfTest.java
  ```

  Arrays.copyOf()实现的是数组的值拷贝还是地址拷贝？

  首先，拷贝出的**新数组和原数组地址不相同**；

  当被拷贝的数组元素是基本的数据类型时，实现的就是值拷贝；

  当被拷贝的数组元素是不是基本的数据类型时，即引用类型，实现的就是地址拷贝；
